# Video Tools

## [VLC](https://www.videolan.org/index.html)

VLC is the best video player for any platform

::: code-group

```sh [Arch]
sudo pacman -S vlc
```

```sh [Debian]
sudo apt update &&sudo apt install vlc
```

```sh [Fedora]
sudo dnf install vlc
```

```sh [Ubuntu]
sudo apt update
sudo apt install vlc
```

```sh [Void]
sudo xbps-install vlc
```

:::
